# TMDBClient

A TMDB Mobile App made using MVVM Architecture

Features:
1) Provides Popular Artists Cards with regular Updates
2) Popular Movies Cards with regular Updates
3) Popular TV Shows Cards with regular Updates


<b>FrameWork Used:</b>

  1) Dagger 2
  2) Cororutines
  3) DataBinding
  4) LiveData
  5) ROOM
  6) Retrofit
